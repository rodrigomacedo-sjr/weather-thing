# TOP Weather App

Part of [The Odin Project's](https://www.theodinproject.com) JavaScript course.
The goal is to work with asynchronous code using Promisses and `async`/`await`.

## Assignment
- [x] Setup boilerplate
- [x] Write functions that hit the api
- [ ] Write function that process the data from the API into an object
- [ ] Set up a form for users to input their location
- [ ] Display the info on the webpage
- [ ] Display weather icons
- [ ] Add basic styling
- [ ] Add a loading component
- [ ] Publish on GH
